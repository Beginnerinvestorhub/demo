# Contributing to Beginner Investor Hub

Thank you for your interest in contributing! We welcome contributions of all types, including bug fixes, features, documentation, and ideas.

## How to Contribute
1. **Fork the repo** and create a new branch for your changes.
2. Make your changes and commit them with descriptive messages.
3. Submit a **pull request (PR)** explaining your changes.

## Guidelines
- Follow the coding style used in the repo.
- Add comments where helpful.
- Keep PRs focused on one topic for easier review.

## Reporting Issues
If you find a bug or have a suggestion, please open an **issue** describing the problem or idea in detail.

Thank you for helping make Beginner Investor Hub better!
