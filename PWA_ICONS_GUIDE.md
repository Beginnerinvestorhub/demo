Visit: https://www.pwabuilder.com/imageGenerator

Upload your logo and it will generate all required sizes including:

- icon-512.png (512x512)
- favicon.ico
- Apple touch icons
- And more...

## Manual Creation (If you have design skills)

### Required Sizes:
```bash
# Create these PNG files in public/ directory:
public/icon-192.png  # 192x192px
public/icon-512.png  # 512x512px
```

### Design Guidelines:
- **Simple logo** on transparent background
- **High contrast** for visibility
- **Minimal text** - scales poorly
- **Square format** - no rounded corners

## Fastest Fix (5 minutes)

1. Go to https://favicon.io/favicon-generator/
2. Upload any image/logo
3. Download the generated icons
4. Copy `android-chrome-192x192.png` → `icon-192.png`
5. Copy `android-chrome-512x512.png` → `icon-512.png`
6. Place both in `public/` folder
